# Website Redesign Plan — Brighter, Warmer, Static Libraries

## Overview
Transform the dark cafe website into a brighter, warmer, more inviting design. Make all libraries static/local. Simplify admin panel. Integrate logo files. Improve readability.

---

## 1. Download Static Libraries & Fonts

Create `fonts/` and `lib/` directories. Download:

### Fonts (`fonts/`)
- Download Vazirmatn font files (woff2) for weights: 300, 400, 500, 700, 800, 900
- Source: Google Fonts / GitHub (font-face in CSS points to local files)

### Libraries (`lib/`)
- `lib/tailwindcss.js` — Tailwind CSS v3 Play CDN (download the script)
- `lib/alpine.min.js` — Alpine.js v3.x CDN minified
- `lib/supabase.min.js` — @supabase/supabase-js v2 UMD bundle

## 4. Static Library References

### New directory structure
```
lib/
├── tailwindcss.js      (Tailwind CSS Play CDN v3)
├── alpine.min.js       (Alpine.js v3.x)
└── supabase.min.js     (Supabase JS v2 UMD)
fonts/
├── Vazirmatn-Thin.woff2
├── Vazirmatn-Light.woff2
├── Vazirmatn-Regular.woff2
├── Vazirmatn-Medium.woff2
├── Vazirmatn-Bold.woff2
├── Vazirmatn-ExtraBold.woff2
└── Vazirmatn-Black.woff2
```

### HTML changes
Replace all CDN references with local paths:
```html
<!-- Before -->
<script src="https://cdn.tailwindcss.com"></script>
<script defer src="https://cdn.jsdelivr.net/npm/alpinejs@3.x.x/dist/cdn.min.js"></script>
<script src="https://cdn.jsdelivr.net/npm/@supabase/supabase-js@2"></script>
<link href="https://fonts.googleapis.com/css2?family=Vazirmatn:...">

<!-- After -->
<link rel="stylesheet" href="fonts/fonts.css">
<script src="lib/tailwindcss.js"></script>
<script defer src="lib/alpine.min.js"></script>
<script src="lib/supabase.min.js"></script>
```

### Font CSS (`fonts/fonts.css`)
```css
@font-face {
  font-family: 'Vazirmatn';
  src: url('Vazirmatn-Regular.woff2') format('woff2');
  font-weight: 400;
  font-style: normal;
  font-display: swap;
}
/* ... repeat for each weight */
```
