Four updates to the کافه آی‌چای website while maintaining all existing improvements.

## 1. Light/Dark Mode Toggle (Public Page)

**Files:** `index.html`, `css/main.css`, `js/app.js`

### CSS (`css/main.css`)
- Add `.dark-mode` CSS block after `:root` that overrides all CSS custom properties with dark equivalents:
  - `--cafe-bg: #1a1410` (deep brown-black)
  - `--cafe-card-bg: #241c14` (dark surface)
  - `--cafe-text: #f0ebe2` (light text)
  - `--cafe-text-muted: #a89880` (muted light)
  - `--cafe-border: rgba(184,134,11,0.15)` (subtle gold border)
  - `--cafe-footer-bg: #1a1410` (match bg)
  - `--cafe-brown: #2d2318` (dark skeleton)
  - `--cafe-black: #faf6f0` (inverted — light text for dark bg)
  - `--cafe-section-bg` for alternating sections
- Update `.navbar` background for dark mode (dark translucent)
- Update `.search-input` for dark mode
- Update `.chip` for dark mode
- Update `.hero` stays dark in both modes (already dark)
- Update skeleton shimmer for dark mode
- Update selection color for dark mode

### JavaScript (`js/app.js`)
- Add `darkMode: false` state
- Add `initDarkMode()` method: read `cafe_dark_mode` from localStorage, apply/remove `.dark-mode` class on body
- Add `toggleDarkMode()` method: toggle state, persist to localStorage, toggle class on body
- Call `initDarkMode()` in `init()`

### HTML (`index.html`)
- Add dark mode toggle button in navbar (desktop): moon/sun SVG icon, placed next to favorites badge
- Add dark mode toggle in mobile menu as well
- Both use `@click="toggleDarkMode()"`

## 2. Favorites / Wishlist Tab

**Files:** `index.html`, `css/main.css`, `js/app.js`

### JavaScript (`js/app.js`)
- Add `showFavoritesOnly: false` state
- Add `get displayProducts()`:
  - If `showFavoritesOnly === true`, return `this.products.filter(p => this.favorites.includes(p.id))`
  - Otherwise return `this.filteredProducts` (existing logic)
- Modify the menu section to use `displayProducts` instead of `filteredProducts`
- Add `toggleFavoritesView()` method: toggles `showFavoritesOnly`, resets category to `cat-1` and clears search when activating favorites view

### HTML (`index.html`)
- Add "علاقه‌مندی‌ها" button in desktop nav (between "ویژه‌ها" and "درباره ما") — heart icon + text, styled with red accent when active
- Add same in mobile nav
- Update menu section: change `x-for="product in filteredProducts"` to `x-for="product in displayProducts"`
- Add favorites empty state: when `showFavoritesOnly && displayProducts.length === 0`, show heart icon + "هنوز محصولی به علاقه‌مندی‌ها اضافه نشده" message + "مشاهده منو" reset button
- Make the existing navbar heart badge clickable to toggle favorites view

### CSS (`css/main.css`)
- Style `.favorites-btn` (nav button) — red accent when `.active`
- Style favorites empty state
